# Stage 7 大模型评估与数据闭环学习

这个文件夹不是训练产物，也不是让你立刻跑模型。它是 Stage 7 的学习入口，目标是先把“大模型怎么评估、评估结果怎么指导数据、微调后怎么验收”讲清楚。

建议顺序：

```text
00_先理解大模型评估_不是先跑分.ipynb
01_评测集怎么设计_从能力维度到样本.ipynb
02_怎么跑baseline_为什么要多路对比.ipynb
03_怎么打分和看badcase_别盲信一个分数.ipynb
04_微调后怎么验收_提升与回归保护.ipynb
```

学习路线：

```text
先理解评估是什么
  -> 再设计评测集
  -> 再跑 baseline
  -> 再看 badcase
  -> 再决定补什么 SFT / DPO 数据
  -> 最后训练后做目标能力评估和旧能力回归保护
```

重要原则：

- 评估不是“让模型自己给自己打分”。
- 生成回答的是模型；打分可以来自规则、人类复核、外部 judge、公开 benchmark 或多种信号组合。
- 面试项目里最重要的是可解释闭环：哪里差，为什么差，补什么数据，补完怎么证明变好，有没有把旧能力训坏。
